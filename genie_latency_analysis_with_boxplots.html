⚙️ How Total Time is Calculated

The total duration for each Genie tool call is calculated using the following formula:

total_duration = asking_ai + pending_warehouse + Execution

Note: The Execution column represents max(sql_execution, executing_query) because these represent alternative execution paths, not nested stages. When both exist, only the larger value is counted to avoid double-counting the query execution time.
